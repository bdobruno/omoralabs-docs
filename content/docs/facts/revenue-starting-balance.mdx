export const caption = "Revenue Starting Balance"
export const table_name="revenue_starting_balance"

## Facts
### Definitions

| Column | Description |
|--------|-------------|
| date | Date in YYYY-MM-DD format, reference to [Dates](/semantic-layers/dates). |
| value_type_id | Type of value measured (links to [Value Types](/semantic-layers/value-types), e.g., actuals, budget). |
| nr_of_customers | Volume variable with number of customers contributing to this revenue change. |
| value_per_customer | Pricing variable, with revenue per customer for this change. |
