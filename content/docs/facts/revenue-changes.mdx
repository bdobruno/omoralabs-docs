## Facts
### Definitions

| Column | Description |
|--------|-------------|
| period_id | Factual reference to the period (links to [Periods](semantic-layer/periods)). |
| revenue_type_id | Type of revenue change (links to [Revenue Types](semantic-layer/revenue-types), e.g., New Booking, Upsell). |
| value_type_id | Type of value measured (links to [Value Types](semantic-layer/value-types), e.g., actuals, budget). |
| nr_of_customers | Volume variable with number of customers contributing to this revenue change. |
| value_per_customer | Pricing variable, with revenue per customer for this change. |
