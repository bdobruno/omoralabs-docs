## Semantic Contract
### Definitions
| Column | Description |
|--------|-------------|
| id | Unique identifier for the GL account. |
| name | Name of the GL account (e.g., Revenue, Cost of Goods Sold, Marketing Expense). |
| parent_gl | Self-reference to the GL account for hierarchy (nullable if top-level). |
| gl_account_type_id | Reference to the [GL Account Type](./gl-accounts-types). |
| business_unit_id | Reference to the business unit (only if segmented; links to [Business Units](./business-units)). |
