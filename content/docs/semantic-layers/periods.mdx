### Definitions

| Column | Description |
|--------|-------------|
| date | Actual calendar date. |
| year_month | Monthly period in YYYYMM format (e.g., 202601 for January 2026). |
| year_week | Weekly period in YYYYWW format using ISO week numbering (e.g., 202552 for week 52 of 2025). |
